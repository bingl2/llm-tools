# /timebox-now - 지금 뭐하지?

회의 다녀왔거나, 잠깐 자리 비웠거나, 문득 "나 지금 뭐해야 하지?" 할 때 쓰는 커맨드.
현재 시각 기준으로 지금 블록과 다음 할 일을 안내하고, 짧은 체크인을 진행합니다.

> loop는 자동 알림(출력만), now는 대화형 체크인(AskUserQuestion 사용)입니다.

## 경로
`$TIMEBOX_HOME` (미설정 시 `~/timebox`). 마스터 파일은 Read → Edit만 (Write 금지).

## 실행 흐름

### 1단계: 현재 상태 파악

1. `date +%H:%M`으로 현재 시각 확인
2. 오늘 타임박스 파일 Read
   - 경로: `{base}/plans/{오늘날짜}.md`
   - **파일 없으면**: "`/timebox`로 오늘 플랜을 먼저 생성해주세요." 안내 후 종료
3. 오늘 로그 Glob: `{base}/logs/{오늘날짜}/*.md` — ad-hoc 카운트 및 방향 이탈 감지용 (있으면)
4. Schedule 섹션을 파싱하여 현재 시각이 어느 블록에 해당하는지 판단

### 2단계: 현재 블록 체크인

**블록 진행 중일 때**:
현재 블록 정보를 보여주고 빠른 체크인:

```
## 현재 블록: 09:00-10:30 | Deep Work
**Focus**: API 리팩토링
- [x] 엔드포인트 정리
- [ ] 에러 핸들링 추가  ← 지금 여기

⏱️ 남은 시간: 42분

> 빠른 체크인 (선택):
> 1. 순조롭다 → 계속
> 2. 막혔다 → 도움 필요
> 3. 블록 연장 → 다음 블록 밀기
> 4. 일찍 끝남 → 다음 블록으로
```

- AskUserQuestion으로 체크인 응답을 받습니다 (스킵 가능)
- 응답에 따라:
  - **순조롭다**: "좋습니다! 남은 {n}분 동안 {작업}에 집중하세요."
  - **막혔다**: 어떤 부분에서 막혔는지 간단히 묻고 방향 제안
  - **블록 연장**: 다음 블록 시작시간을 조정 (마스터 파일 Edit)
  - **일찍 끝남**: 현재 블록 태스크를 [x]로 마킹하고 다음 블록 안내

**방향 이탈 감지:**
- 사용자가 Big 3와 무관한 작업을 하고 있으면:
  "지금 하시는 건 Big 3에 없는 작업입니다. 의도적 선택인가요, 아니면 빠져든 건가요?"
- ad-hoc이 2건 이상 쌓이면:
  "오늘 ad-hoc이 {n}건입니다. Big 3 중 {미완료}가 위험해지고 있습니다."
- 블록 타입과 작업 불일치:
  "지금은 Shallow 시간인데 Deep Work을 하고 계십니다. 의도적이면 블록을 변경할까요?"

**블록 전환 시점(Break)일 때**:
```
## 블록 전환: Break (10:30-10:45)

### 이전 블록 리뷰
- Deep Work (09:00-10:30): {Focus}
- 완료: {n}/{total} 작업

### 다음 블록 프리뷰
- 10:45-12:00 | Deep Work
- Focus: {Big 3 #2}
- 예정 작업: {목록}

> 조정하시겠습니까?
> 1. 그대로 진행
> 2. Focus 변경
> 3. 이전 블록 이어하기
```

**근무 시간 이후일 때**:
- "오늘 업무 시간이 지났습니다."
- Big 3 진행률 보여줌
- "`/timebox-end`로 하루를 마무리하시겠습니까?" 안내

### 3단계: 에너지 로그 (선택)

체크인 시 에너지 레벨을 물어봅니다 (빠르게, 1-5 또는 스킵):
- 응답하면 마스터 파일의 Energy Log 테이블에 Edit으로 추가:

```
| 10:30 | 4/5 | 집중 잘 됨, 다음 블록도 OK |
```

### 4단계: 마스터 파일 업데이트

- 완료된 작업이 있으면 `- [ ]` → `- [x]` 변환 (Edit)
- 블록 조정이 있으면 시간 수정 (Edit)
- **반드시 Read 후 Edit 사용. Write 금지.**

## 참고

- 마스터 파일: `{base}/plans/{YYYY-MM-DD}.md`
- 블록 시간 파싱: `### HH:MM-HH:MM | 블록타입` 패턴으로 매칭
- 에너지 체크인은 선택사항, 스킵 가능
- 체크인은 30초-1분 이내로 빠르게 진행
